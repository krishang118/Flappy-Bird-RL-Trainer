# Flappy Bird Q-Learning (RL) Trainer

A Q-learningâ€“based reinforcement learning agent that learns to play the Flappy Bird game using Pygame. The agent improves over time through trial-and-error interactions with the game environment.

## Requirements

- Python 3.x
- Pygame
- NumPy

Install the dependencies with:
```bash
pip install pygame numpy
```

## Usage

Run the training script:
```bash
python "Flappy Bird Training.py"
```

The agent will start learning to play Flappy Bird. The game window will display the current score, episode, epsilon (exploration rate), and best score achieved.
